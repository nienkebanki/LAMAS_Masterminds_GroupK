Setup

  • Two agents: $A$ (user) and $B$ (computer).
  • Code length 3, colors $\{0,1,2\}$, no repetition.
  • $A$ knows $c_A$, $B$ knows $c_B$.
  • The user does not know $c_B$ (default mode).

Move and feedback

Each round, $A$ chooses a guess $g$. The system publicly reveals two feedback values: $$F_B = fb(g,c_B)\quad\text{and}\quad F_A = fb(g,c_A).$$

Feedback function: standard Mastermind score $(b,w)$ where $b$ counts correct color+position and $w$ counts correct color in a different position.

Winning condition

Classical win: $A$ guesses $c_B$ exactly. Epistemic win: $A$ wins as soon as $A$ knows $c_B$. In the Kripke model this is: $$|Cand_A(B)|=1.$$

Edge case: same code

If $c_A=c_B$, then for every guess $g$ we always observe $F_A=F_B$. This becomes public information and can speed up identification. Our implementation allows this case.
