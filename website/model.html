Worlds

A world is a pair $(c_A,c_B)$. With 3 colors and length 3 without repetition, each agent has $3!=6$ possible codes, so initially there are $6\cdot 6=36$ worlds.

Accessibility relations

Agent $A$ knows their own code, so they cannot distinguish worlds with different $c_A$: $$(c_A,c_B)\sim_A(c'_A,c'_B)\ \text{iff}\ c_A=c'_A.$$ Similarly, $$(c_A,c_B)\sim_B(c'_A,c'_B)\ \text{iff}\ c_B=c'_B.$$

These are equivalence relations (S5).

Candidate sets

$Cand_A(B)$ is the set of $c_B$ values appearing in $A$’s information set. $A$ knows $B$’s code exactly when $|Cand_A(B)|=1$.
